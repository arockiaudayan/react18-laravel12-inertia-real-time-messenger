# 💬 Real-Time Messenger App

A blazing fast real-time messaging application built using **Laravel 12**, **React 18**, **Inertia.js**, and **WebSockets**.

## 🧱 Tech Stack

* ✅ Laravel 12 (API Backend)
* ⚛️ React 18 + Inertia.js (Frontend)
* 🔐 Laravel Sanctum (Authentication)
* 📡 WebSockets (Pusher or Laravel Echo)
* 🎨 TailwindCSS
* 🧩 Vite (Frontend build tool)

---

## 🚀 Features

* Real-time chat with WebSocket support
* Typing indicators
* Seen/delivered message status
* Online/offline user presence
* Authentication and roles
* Responsive design (mobile-ready)
* Clean and modular codebase

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/arockiaudayan/react18-laravel12-inertia-real-time-messenger.git
cd react18-laravel12-inertia-real-time-messenger
```

### 2. Install dependencies

```bash
composer install
npm install
```

### 3. Set up environment variables

```bash
cp .env.example .env
php artisan key:generate
```

### 4. Configure your DB in `.env` then run:

```bash
php artisan migrate
```

### 5. Start the development servers

```bash
npm run dev
php artisan serve
```
