## ğŸ“¦ Installation

### 1. Clone the repository

```bash
git clone https://github.com/arockiaudayan/react18-laravel12-inertia-real-time-messenger.git
cd react18-laravel12-inertia-real-time-messenger
```

### 2. Install dependencies

```bash
composer install
npm install
```

### 3. Set up environment variables

```bash
cp .env.example .env
php artisan key:generate
```

### 4. Configure your DB in `.env` then run:

```bash
php artisan migrate
```

### 5. Start the development servers

```bash
npm run dev
php artisan serve
```
